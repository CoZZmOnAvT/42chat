# 42Chat
Socket based chat sever & client.

You can build this project on MacOS, Ubuntu/Debian, Android.

Copilation:
  - `bash ./configure.sh`
  - `make`

Execution:
  - If you want to start your own chat server:
    - `./chat_server`
    - `./42chat [YOUR_IP_ADDRESS]`
  - If you want to chat in the my dedicated server:
    - `./42chat 91.203.25.95`

Tested systems:
  - MacOS High Sierra
  - Ubuntu 16.04 & 18.04
  - Android 7.0 & 8.1
